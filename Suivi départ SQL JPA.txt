Création packages :

App (main)
   ↓
Service (logique métier)
   ↓
Repository (accès base de données)
   ↓
Entity (objets liés aux tables)
   ↓
Base de données MySQL

--------------------------------------------------------------------------------------------------
---- entity contient les classes JPA, chaque classe = une table SQL


---- repository sert à intéragir avec la bdd

on y met insert select update delete ect ect

----- service contient la logique de l'appli

Exemple dans le projet :

vérifier si un prof peut déplacer un cours

accepter/refuser une demande

récupérer l’emploi du temps d’un élève

---- config contient la config de l'app

- connexion base de données

- configuration JPA

- configuration Hibernate
